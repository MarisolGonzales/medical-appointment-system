# Medical Appointment System

This is a university project built with Java and Spring Boot.

This project is a proposal to improve Essalud's medical appointment system in Peru. Essalud is a public Peruvian healthcare institution that currently faces efficiency problems in its appointment scheduling system. Many people struggle to book medical appointments due to these issues. 

The goal of this project is to develop a more efficient and user-friendly solution to help Peruvians access medical appointments more easily.
