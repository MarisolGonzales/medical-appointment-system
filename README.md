A university project developed to improve the medical appointment management system of Essalud (Peru's public healthcare institution).

## 📌 Description

This project proposes a more efficient and user-friendly solution for scheduling and managing medical appointments. Essalud currently faces challenges with its appointment system, making it difficult for many Peruvians to access medical consultations. This application aims to address those issues through a cleaner and more organized platform.

## ✨ Features

- User registration and authentication
- Medical appointment scheduling
- Appointment management (view, cancel, reschedule)
- Doctor and specialty management
- Basic admin panel (future implementation)

## 🛠 Technologies Used

- **Java 21+**
- **Spring Boot**
- **Spring Data JPA**
- **Maven**
- **Thymeleaf** (or React if frontend is added)
- **MySQL / PostgreSQL**

## ▶️ How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/MarisolGonzales/medical-appointment-system.git

2. Open the project in your IDE (IntelliJ, Eclipse, VS Code, etc.)
3. Configure your database connection in application.properties
4. Run the application:
   ./mvnw spring-boot:run
